```
devlink dev show # 显示所有 devlink 设备的信息
devlink dev reload pci/...  # 重新加载指定的 devlink 设备
devlink dev info pci/... # 显示指定 devlink 设备的详细信息
devlink dev param show pci/... # 显示指定 devlink 设备的参数
devlink port show pci/...  # 显示指定 devlink 设备的端口信息
devlink port set pci/... rate 100000  # 设置指定 devlink 设备端口的速率为 100000 Mbps
devlink resource show pci/...  # 显示指定 devlink 设备的资源信息
devlink health show pci/...  # 显示指定 devlink 设备的健康状态
devlink monitor start pci/...  # 启动对指定 devlink 设备的监控
devlink monitor stop pci/...  # 停止对指定 devlink 设备的监
```
