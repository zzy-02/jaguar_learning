一个文件的属性以及文件所属的用户和组以及权限：
![](图片\文件字符信息对应.jpeg)
- 第一位字符file type:
```
当为 d 则是目录
当为 - 则是文件；
若是 l 则表示为链接文档(link file)；
若是 b 则表示为装置文件里面的可供储存的接口设备(可随机存取装置)；
若是 c 则表示为装置文件里面的串行端口设备，例如键盘、鼠标(一次性读取装置)。
```
- 接下来的字符中，以三个为一组，且均为 `rwx` 的三个参数的组合。其中，
    - `r` 代表可读(read)
    - `w` 代表可写(write) 
    - `x` 代表可执行(execute)
要注意的是，这三个权限的位置不会改变，如果没有权限，就会出现减号 `-` 而已。

#### 着重介绍chmod
- 权限分数对照如下：
    - r:4
    - w:2
    - x:1
例如：权限为-rwxrwx---，则权限数字为770
owner = rwx = 4+2+1 = 7
group = rwx = 4+2+1 = 7
others= --- = 0+0+0 = 0

- 变更权限的语法：
 `chmod [-R] xyz 文件或目录`
 `xyz` : 就是刚刚提到的数字类型的权限属性，为 rwx 属性数值的相加。
 `-R` : 进行递归(recursive)的持续变更，以及连同次目录下的所有文件都会变更

- 其他：`chmod +x 文件名`也比较常用，加可执行权限
